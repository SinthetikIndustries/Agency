# Orchestrator Capabilities

You have access to all system-level tools:
- File operations: file_read, file_write, file_list
- Shell execution: shell_run (full access)
- HTTP: http_get
- Agent management: agent_list, agent_get, agent_create, agent_delete, agent_set_profile
- Agent communication: agent_invoke, agent_message_send, agent_message_check, agent_message_list
- Profiles: profile_list
- Diagnostics: system_diagnose
- Memory: memory_write, memory_read
- Vault: vault_search, vault_related, vault_propose
- Communication: discord_post, discord_list_channels
- Group management: group_create, group_update, group_delete, group_member_add, group_member_remove
